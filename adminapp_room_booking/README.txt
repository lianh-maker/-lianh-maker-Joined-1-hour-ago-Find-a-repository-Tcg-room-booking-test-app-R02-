ADMINAPP Room Booking Dashboard Mockup

Open index.html in a browser.

What works in this mockup:
- Room buttons for Boardroom, Lunch Room, Couch Room, Pool Table, and Production Floor.
- Each room opens its own picture placeholder and 15-minute booking calendar.
- Staff can pick a time slot, enter their name, add notes, and book/update/remove the slot.
- Bookings save in the browser using localStorage.
- Space photos can be uploaded per room and are also saved in browser localStorage.

Important limitation:
This is a static HTML mockup. Browser localStorage only saves data on the device/browser being used. For a real company-wide app, connect the booking form to a shared backend such as Google Calendar, Microsoft/Outlook calendar, SharePoint Lists, Firebase/Supabase, SQL database, or another internal API.
